<p><strong style="font-size:24px;">Upgrading a RAM card in Laptop Device</strong></p>


## Purpose 
This document provides the procedure for replacing or upgrading a laptop's RAM module and verifying the installation in Windows.

## Before You Begin
* Verify the replacement RAM is compatible with the laptop.
* Shut down the laptop completely.
* Disconnect the power adapter.
* Disconnect all external peripherals.
* Ground yourself to prevent electrostatic discharge (ESD).

## Tools and Software Needed: 
- Electronic Toolset
- Laptop
- Windows 10/11 OS 
- Task Manager
- Compatible RAM Module

## Prerequsites
* Compatible replacement RAM module.
* Laptop supports memory upgrades.
* Laptop is powered off.
* Battery is disconnected (if removable).
* Administrator access to Windows (for verification).

## Environment
 Windows 10/11 OS

## Cause 
User may want to upgrade Laptop RAM due to laptop performance issues 

![Task Manager thrashing disk](task-manager-disk-thrash-1.PNG)

![Task Manager thrashing disk](OLD-RAM-SS-1.png)


## Procedure

1. First safely shutdown the laptop 

2. Use a screwdriver to remove the screws in the back panel Remove the back panel from the laptop

3. Release the retaining clips on both sides of the RAM module. The module should rise to approximately a 45° angle.

4. Remove the RAM module out of the slot and the new one replaced at the same 45 degree angle

5. Insert the replacement RAM module at approximately a 45° angle. Press downward until both retaining clips lock into place.

6. Place the back panel and put back the screws 

7. Turn on the Laptop as needed

8. Verify on Task Manager or Systems Information that the RAM space has been updated
![RAM addition](disk.change.1-1.PNG)


#### Refer to RAM Upgrade Demonstration for video demonstration

## Verification
* Task Manager reports the expected installed memory.
* Windows boots successfully.
* No memory-related errors are reported.
* Applications launch normally after the upgrade.

## Extra Notes:
If the laptop does not recognize the new memory:
* Reseat the RAM module.
* Verify compatibility with the laptop model.
* Confirm the RAM is fully inserted into the slot.
