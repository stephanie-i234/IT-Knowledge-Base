# IT Support Knowledge Base

## Overview

This repository contains a collection of technical documentation created while completing hands-on IT support projects and laboratory exercises. The knowledge base is intended to demonstrate technical writing skills, troubleshooting methodology, and the ability to document repeatable IT procedures in a clear and structured format.

The documentation covers common IT support tasks involving Windows Server, Active Directory, Windows 10/11, VMware Workstation, networking, virtualization, and basic computer hardware.

---

```text

│
├── KB-001 Hardware

```

Each knowledge base section contains:

* Technical documentation (`.md`)
* Supporting screenshots (where applicable)
* Related video demonstrations (where applicable)
